Root

En el root del archivo se encuentran las variables css donde podemos definir estilos para nuestro tema.

            
    /*Completar con la paleta del proyecto*/
    --color-primario: ;
    --color-secundario: ;
    --color-terciario: ;
    --color-asento-1: ;
    --color-asento-2: ;
    --color-efecto-1: ;
    --color-efecto-2: ;
    --color-text-base: ;
    --color-text-referencias: ;

    /*Completar con las medidas estandar de las tarjetas del proyecto en px*/
    --padding-card: ;
    --padding-wide-card: ;

    /*Completar con las medidas de los iconos del proyecto en px*/
    --icon-small: ;
    --icon-medium: ;
    --icon-large: ;

    /*Completa con el grosor de subrayado o hr que usas en el proyecto en px*/
    --grosor-hr: ;

    /*Completar con el tiempo de transición para el cambio de color de los textos al hacer hover*/
    --tiempo-de-transicion-color-texto: ;

    /*Completar con el tiempo de transición para el cambio de color de los background al hacer hover*/
    --tiempo-de-transicion-color-bg: ;

Switch elements responsive

Ocultar / mostrar componentes en distintos dispositivos

                   

    @media (min-width:991px) {
        .ocultar-en-pc {
            display: none;
        }
    }
    
    @media (max-width:991px) and (min-width: 768px) {
        .ocultar-en-tablet {
            display: none;
        }
    }
    
    @media (max-width:768px) {
        .ocultar-en-mobile {
            display: none;
        }
    }
